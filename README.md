# DesafioMobile2You
Aplicativo de teste para a empresa Mobile2You

## Descrição:

O aplicativo foi desenvolvido de forma nativa, com a IDE Android Studio e a linguagem de programação Kotlin.

O projeto foi desenvolvido com a arquitetura MVVM.

Foi utilizada a API do "The Movie DB" para adquirir informações dos filmes.

Foram utilizados Retrofit e Moshi para requisição dos dados e conversão do Json.

Neste projeto foram utilizados testes unitários para as funções de abreviação de número de likes e requisição de gêneros.
